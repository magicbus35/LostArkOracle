# 프로젝트 기록: 로스트아크 마켓 오라클 (LMO)

## 2026-03-16: 크롤러 자동화 및 AI 추론 UI 강화 (Explainable AI)

### 1. 통합 자동화 파이프라인 (Auto-Trigger System) 구축
- **작업 내용**: 공식 홈페이지 `crawl_notices.py` 크롤러 내부 로직에 `subprocess` 모듈을 연동하여 자율 트리거 시스템 도입.
- **효과**: 신규 공지사항이 단 1건이라도 탐지될 경우, 시스템이 스스로 후속 파이프라인(`데이터 수집` -> `LLM 경제 분석` -> `시장 상승/하락분 피처 추출` -> `XGBoost 예측 모델 재학습`)을 연쇄적으로 호출. 사용자 개입 없이 전체 프로세스가 1~2분 내에 자동 완주되도록 **완전 무인화(Full Automation)** 달성.

### 2. Streamlit 대시보드 UI/UX 고도화 및 XAI 도입
- **AI 산출 근거 시각화**: 기존에는 숫자 형태의 예측 차트만 보여주었으나, 이제는 섹터별 예측 그래프 하단에 인공지능이 왜 그런 차트 궤적을 예상했는지(`mechanisms_applied` 분석 원문과 파라미터 점수)를 명시적으로 출력하는 전용 컨테이너 박스 신설.
- **LLM 한국어 출력 강제화**: `analyze_events_llm.py` 백엔드 프롬프트를 수정하여, AI가 판독한 경제 분석 논리와 근거를 무조건 한국어로 작성하여 리턴하도록 프롬프트 엔지니어링 수행.
- **번역 필터링 버그 Fix**: 대시보드의 영문 태그('Stream Summary', 'Roadmap Item')를 한국어('방송 요약', '로드맵 예고')로 사용자 친화적으로 번역. 번역 직후 발생했던 내부 인벤 탭 검색 필터링 충돌 오류를 다중 조건(`isin`) 마스킹으로 깔끔하게 디버깅 완수.
- **종합 효과**: 투자자(유저)가 블랙박스 같던 AI의 판단 근거(재련 재료 수요/공급 가중치 등)를 차트와 동시에 읽을 수 있게 되어, 퀀트 투자의 핵심인 **모델 신뢰도(Trust & Transparency)**가 대폭 격상됨.

## 2026-01-16: 이벤트 데이터 강화 및 모델링

### 목표
가격 예측 모델의 성능을 향상시키기 위해 공식 이벤트 데이터에 시장 영향력 분석을 추가합니다.

### 수행한 작업
1.  **데이터 수집 (Scraping)**
    - `scripts/scrape_notice_content.py` 생성.
    - 450개 이상의 공식 로스트아크 공지사항에서 전체 HTML 본문 스크래핑.
    - 본문 추출을 위한 CSS 선택자 `div.fr-view` 식별.

2.  **LLM 데이터 강화 (Perplexity API)**
    - `scripts/analyze_events_llm.py` 생성.
    - Perplexity (`sonar-small-online`)를 사용하여 공지사항 텍스트 분석.
    - 불리언 플래그 추출: `골드 너프(Gold Nerf)`, `신규 레이드(New Raid)`, `T4 업데이트`, `보석 공급(Gem Supply)`, `신규 패키지`.
    - 결과 저장: `data/events_enriched.csv`.

3.  **피처 엔지니어링**
    - `src/process_data.py` 업데이트.
    - 추출된 플래그를 메인 학습 데이터셋(`lostark_features.csv`)에 통합.
    - 생성된 피처: `is_gold_nerf`, `is_new_raid`, `is_t4`, `is_gem`, `is_package`.

4.  **모델 개선**
    - 신규 피처를 포함하도록 `scripts/train_model.py` 업데이트.
    - XGBoost Regressor 재학습.
    - **결과**: RMSE(평균 제곱근 오차)가 **58,987**에서 **57,869**로 개선됨 (~1.9% 향상).

- **`scripts/enrichment`**: 데이터 강화 (`analyze_events_llm.py`).
- **`scripts/modeling`**: 모델 로직 (`train_model.py`).
- **`scripts/ops`**: 운영 및 유지보수 (`news_watcher.py`).

### 7. 미래 데이터 누수 차단 및 모델 검증 (2026-01 ~ 2026-02)
- **비순환 롤링 윈도우 (Walk-Forward Validation)**: 과거 이벤트를 교차 검증할 때 미래 데이터를 훔쳐보는(Data Leakage) 심각한 오버피팅 오류를 발견. 이전 시점의 데이터로만 모델을 구축하도록 제어하는 "Honest Prediction" 아키텍처를 도입하여 AI 예측의 신뢰도 확보.

### 8. 투-트랙 수집 아키텍처 구축 (인벤 커뮤니티 연동)
- **커뮤니티 심리 분석**: 공식 패치노트의 한계를 극복하기 위해 `crawl_inven.py` 개발.
- **필터링 로직**: 단순 뻘글이 아닌, 요약 전문 유저('가지있는나무' 등)의 10추글을 우선 식별하고, 이진 탐색(Binary Search) 기반의 최적화된 페이지 탐색 알고리즘(Smart Year Inference)을 도입하여 방송 직후의 핵심 여론만 정밀 타겟팅.

### 9. 이벤트 텍스트 분석 고도화 (Hybrid Keyword Override)
- **스코어 민감도 조정**: 1~2월 대형 업데이트 당시 시장 수요가 70% 폭증했음에도 LLM이 뉘앙스 파악에 실패해 점수를 0점으로 주는 치명적 에러 발견.
- **하이브리드 로직**: 파이썬 단말에서 특정 핵심 트리거(보상, 복구 등)의 출현 빈도를 감지해 LLM의 환각을 방어하고 High Score로 강제 보정하는 2중 안전장치 설계. 예측 그래프의 평탄화 현상(Flatline) 완벽 해결.

### 10. 다중 시간 지평(Multi-Horizon) 궤적 예측 도입
- 단순 1일 뒤 가격 예측이 아닌, 이벤트 발생일(T=0)을 기준으로 T-30일부터 T+30일까지 전체 시세의 궤적(Trajectory)을 한 번에 그려내도록 XGBoost `MultiOutputRegressor` 모델 튜닝.
- 투자자(유저)가 차트를 보며 "T+7에 익절해야 한다"는 인사이트를 직관적으로 얻을 수 있도록 대시보드 시각화 업그레이드.

### 11. 공식 API 연동 및 오버적합/차단 우회 (2026-03)
사설 데이터 사이트 크롤링 한계(IP Ban) 극복 및 포트폴리오화:
- **`scripts/collection/fetch_official_api.py`**: 로스트아크 공식 파트너 API 엔드포인트 연동. 봇 방어(Cloudflare 403 Forbidden)를 우회하여 14일 치 고해상도 시세 데이터를 안정적으로 수급하는 합법적 파이프라인 구축.
- **포트폴리오 초안 완성**: 비순환 롤링 윈도우(미래 참조 방지), LLM 하이브리드 오버라이드, 시계열 궤적(Multi-Horizon) 예측 등 핵심 기술을 문서화(`portfolio_draft.md`).
